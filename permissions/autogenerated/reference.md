## Default Permission

Allows the dev-only Tauri Agent bridge commands for compact semantic trees, semantic ref finding, ref inspection, JavaScript eval, ref actions, hover, focus, blur, scroll, and drag events, select controls, checked controls, screenshots, event and network streams, storage access, and window discovery.

#### This default permission set includes the following:

- `allow-agent-bridge-response`
- `allow-agent-attach`
- `allow-agent-snapshot`
- `allow-agent-find`
- `allow-agent-action`
- `allow-agent-inspect`
- `allow-agent-eval`
- `allow-agent-select`
- `allow-agent-check`
- `allow-agent-hover`
- `allow-agent-focus`
- `allow-agent-blur`
- `allow-agent-scroll`
- `allow-agent-drag`
- `allow-agent-screenshot`
- `allow-agent-logs`
- `allow-agent-events`
- `allow-agent-network`
- `allow-agent-storage`
- `allow-agent-windows`
- `allow-agent-wait`
- `allow-agent-state`
- `allow-agent-record`
